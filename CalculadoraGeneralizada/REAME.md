# 📘 Exercício - Calculadora Funcional em Java

## 📖 Descrição

Este exercício consiste na criação de uma **calculadora funcional em Java**, utilizando uma `ArrayList` como entrada e a **API de Streams** para realizar operações matemáticas sobre múltiplos valores.

A aplicação permite processar uma sequência de números de forma dinâmica, aplicando a operação escolhida de maneira eficiente e escalável.

---

## 🎯 Objetivo

* Aplicar conceitos de ENUM
* Utilizar funções lambda
* Trabalhar com a interface `BiFunction`
* Utilizar a API de Streams
* Praticar organização e reutilização de código

---

## 🧠 Conceitos aplicados

* ENUM
* Funções lambda
* Interface funcional (`BiFunction`)
* API de Streams
* Encapsulamento
* Métodos

---

## 📂 Estrutura do projeto

```bash
📁 src/
 ├── Main.java
 ├── Operacoes.java
 └── RetornoDeNumeros.java
```

---

## ⚙️ Funcionamento

O programa recebe uma sequência de números, converte esses valores para uma lista (`ArrayList`) e utiliza **Streams** para aplicar a operação selecionada.

Essa abordagem permite trabalhar com **quantidade variável de entradas (n valores)**, tornando a solução mais flexível.

---

## ▶️ Como executar

1. Compile o projeto:

```bash
javac *.java
```

2. Execute:

```bash
java Main
```

---

## 💡 Exemplo de uso

**Entrada:**

```
2,2,2,2,2,2
```

**Saída:**

```
12.0
```

---

## 🚀 Aprendizado

* Aprendi a utilizar a API de Streams para processar coleções de dados
* Entendi como aplicar operações em múltiplos valores de forma funcional
* Reforcei o uso de `BiFunction` em conjunto com ENUM
* Melhorei a organização e reutilização do código

---

## 📌 Observação

Este exercício demonstra uma abordagem moderna em Java, combinando **Programação Orientada a Objetos (POO)** com **Programação Funcional**, resultando em um código mais limpo, reutilizável e escalável.
