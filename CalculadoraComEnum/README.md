# 📘 Exercício - Calculadora Funcional em Java

## 📖 Descrição

Este exercício consiste na criação de uma calculadora funcional em Java, utilizando **ENUMs** para representar operações e **funções lambda** para executar os cálculos.

---

## 🎯 Objetivo

* Aplicar conceitos de ENUM
* Utilizar funções lambda
* Trabalhar com a interface `BiFunction`
* Praticar organização e reutilização de código

---

## 🧠 Conceitos aplicados

* ENUM
* Funções lambda
* Interface funcional (`BiFunction`)
* Encapsulamento
* Métodos

---

## 📂 Estrutura do projeto

```bash
📁 src/
 ├── Main.java
 └── OperationEnum.java
```

---

## ▶️ Como executar

1. Compile o projeto:

```bash
javac Main.java
```

2. Execute:

```bash
java Main
```

---

## 💡 Exemplo de uso

**Entrada:**

```
1
10
10
```

**Saída:**

```
O resultado é: 20
```

---

## 🚀 Aprendizado

* Entendi como utilizar `enum` para representar operações fixas
* Aprendi a associar funções lambda a constantes de um ENUM
* Compreendi o uso de `BiFunction` para operações com dois parâmetros
* Melhorei a organização do código com conceitos de POO

---

## 📌 Observação

Este exercício demonstra uma abordagem funcional em Java, combinando conceitos de orientação a objetos com programação funcional.
