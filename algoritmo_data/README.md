# 📘 Exercício - Exibir Data

## 📖 Descrição

Este exercício consiste em exibir a **data atual** utilizando a classe `LocalDate` do Java.  
O programa apresenta a data em dois formatos:

- ISO: `2026-03-24`
- Brasileiro: `24/03/2026`

O objetivo é praticar o uso de **classes da API de Data e Hora** do Java.

---

## 🎯 Objetivo

- Trabalhar com a API `java.time`
- Manipular datas usando `LocalDate`
- Praticar formatação de datas
- Reforçar lógica de programação básica

---

## 🧠 Conceitos aplicados

- `LocalDate`
- Formatação de datas (`DateTimeFormatter`)
- Métodos
- Entrada e saída no console

---

## 📂 Estrutura do projeto

📁 src/ </br>
 └── datahora/ </br>
 
     └── Main.java

---

## ▶️ Como executar

1. Compile todas as classes:

javac *.java

2. Execute o programa:

java Main

---

## 💡 Exemplo de saída

Data no formato ISO: 2026-03-24  
Data no formato brasileiro: 24/03/2026

---

## 🚀 Aprendizado

- Aprendi a trabalhar com datas no Java moderno  
- Entendi como formatar e exibir datas em diferentes padrões  
- Pratiquei organização de classes e métodos  
- Consolidei conceitos de entrada/saída no console  

---

Este exercício é voltado para **iniciantes em Java**, focando na manipulação de datas e boas práticas de programação.  
Ele demonstra uma abordagem limpa e organizada, combinando conceitos de **POO** e **API de Data e Hora do Java**.
