# 📘 Exercício - Sistema de Usuários

## 📖 Descrição

Este exercício consiste em criar um Usuário com login e senha, possibilitar o login, realizar ações na conta e deslogar apagando a conta automaticamente.

O objetivo é praticar conceitos de **POO**, **herança**, **polimorfismo** e **encapsulamento** em Java.

---

## 🎯 Objetivo

- Criar classes para diferentes tipos de usuário (Gerente, Vendedor, Atendente)  
- Trabalhar com login e logout  
- Implementar ações específicas por tipo de usuário  
- Manipular relatórios de vendas e caixa  
- Praticar lógica de programação orientada a objetos  

---

## 🧠 Conceitos aplicados

- Classes e objetos  
- Herança  
- Polimorfismo  
- Encapsulamento  
- Entrada e saída de dados no console  

---

## ▶️ Como executar

1. Compile todas as classes:

```bash
javac *.java
```
Execute o programa:
```
java Main.java
```
